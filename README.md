# For a Traveler(s)
* HTML, CSS, JavaScript 를 이용한 첫 번째 프로젝트입니다.

## 1. 목표와 기능
### 1.1 목표
* 1인 여행객을 위한 웹 페이지 제작
* 추후 여행 물품 판매 페이지, 커뮤니티 페이지 등 페이지 추가를 위한 웹 페이지 디자인
* 1인 여행 플래너 chatbot 제작을 위한 구축된 API 수정
### 1.2 기능
* 1인 여행 플래너 chatbot 제작을 위한 구축된 API 수정
* Javascript 이용하여 API 화면에 렌더링
* 다양한 화면에서도 유저들이 불편함 없이 이용할 수 있도록 반응형 웹 페이지 제작
